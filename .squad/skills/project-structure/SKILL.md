# Skill: Project Structure Scaffolding

Reusable pattern for full-stack web app repo structure:
- frontend/ (React, TypeScript)
- backend/ (ASP.NET Core, MCP)
- infra/ (Terraform, GitHub Actions)
- .squad/ (team, decisions, skills)
- Modular infra: terraform/modules, terraform/environments
- CI/CD: github/workflows
- Rationale: separation of concerns, scalability, onboarding
