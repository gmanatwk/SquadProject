# Technology Stack

## Frontend
- React 18+
- TypeScript 5+
- Redux Toolkit
- React Hook Form

## Backend
- ASP.NET Core (latest, e.g., .NET 8)
- MCP host
- Entity Framework Core
- SQL Server

## Infrastructure
- Terraform (latest)
- GitHub Actions
- Docker (optional, for local/dev)

## Integration Points
- REST API between frontend and backend
- Database migrations via EF Core
- CI/CD via GitHub Actions
- Infrastructure as Code via Terraform
